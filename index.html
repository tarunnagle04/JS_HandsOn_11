Q:7)Write a function that accepts parameter n and returns factorial of n

Answer:

        function factorial(n) {
        if (n === 0 || n === 1)
            return 1;
        for (var i = n - 1; i >= 1; i--) {
            n*= i;
        }
        return n;
        }
        console.log(factorial(5));

Output:120
